## 目录结构

- `modules/public`：首页、搜索、专题、门店等公共浏览页面
- `modules/catalog`：分类、商品列表、商品详情、评价等商品体系页面
- `modules/trade`：购物车、订单、售后等交易链路页面
- `modules/account`：登录注册、会员中心、资产、收藏、设置等账户页面
