# 虚拟订单列表模块

## 对应页面

- `http://www.whhmk.cn/m/tmpl/member/vr_order_list.html`

## 页面结构

- 顶部搜索区
  - 返回按钮
  - 虚拟订单搜索框
  - 虚拟 / 实物切换入口
- 订单状态导航
  - 全部
  - 待付款
  - 待使用
- 虚拟订单列表区
- 订单类型切换浮层
  - 实物订单
  - 虚拟订单
- 支付弹层

## 顶部筛选与搜索

- 搜索占位文案
  - 输入您要搜索的商品名称
- 状态筛选
  - 全部
  - 待付款
  - 待使用
- 实物 / 虚拟切换按钮

## 支付弹层字段

- 在线支付金额
- 充值卡支付选项
- 充值卡余额
- 预存款支付选项
- 预存款余额
- 支付宝选项
- 微信支付选项
- 支付密码输入框
- 去设置支付密码入口
- 确定支付按钮

## 页面可见下级页面入口

- `order_list.html`
- `member_vr_evaluation.html`
- `member_paypwd_step1.html`

## 可见操作

- 搜索虚拟订单
- 按状态筛选虚拟订单
- 切换到实物订单列表
- 发起订单支付
- 取消订单
- 评价订单

## 交互说明（基于页面）

- 点击顶部切换入口展开订单类型选择层
- 选择“实物订单”跳转实物订单列表页
- 点击“订单支付”拉起支付弹层
- 点击“取消订单”弹出确认提示
- 点击“评价订单”进入虚拟订单评价页
- 搜索与状态切换都会刷新订单列表
