# 退货列表模块

## 对应页面

- `http://www.whhmk.cn/m/tmpl/member/member_return.html`

## 页面结构

- 顶部双 Tab
  - 退款列表
  - 退货列表
- 退货记录列表

## 退货记录字段

- 店铺名称
- 退货状态
- 商品图片
- 商品名称
- 商品规格
- 商品单价
- 商品数量
- 退货数量
- 退款金额
- 申请时间

## 卡片操作按钮

- 退款详情
- 延迟
- 退货发货

## 页面可见下级页面入口

- `member_return_info.html`
- `member_return_ship.html`

## 交互说明（基于页面）

- 可切换到退款列表页
- 点击商品卡片或“退款详情”进入退货详情页
- 符合条件时可执行“延迟”
- 符合条件时可进入“退货发货”页
